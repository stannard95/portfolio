Keir Stannard - Portfolio

3D Mobile Trading Game

Created for my dissertation, the game resembles a medieval world entangled in war between four kingdoms. The player must thrive under the world economy, trading goods and funding wars for their own benefit to succeed in the world.

London Game

A game created using Google Maps for an assignment.  It took the London transport network as its “board”, and superimposed a fictional world of monsters and magical items onto the real world of tubes, buses, stations etc.

MVC CRUD app

This is a web application that allows members of the Sparta Global Organisation to upload and view projects they have worked on. Members can make an account, and are then able to create, read, update and delete their projects, and view projects created by other Spartan members.

Escape Sparta

A 2D puzzle escape game based in Sparta Global, the spartan has been locked inside and the power is out. Using their logical and agile mind and they must Escape Sparta.

Posgres Test Environment

This project is a basic test environment containing a Virtual Machine with a Postgres database installed. It is intended for use by those on the Sparta Global SDET course to facilitate the learning process.
